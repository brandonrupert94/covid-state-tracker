State Covid Status Checker

Using this webpage:

Hello and Welcome! During this global pandemic and considering how it is affecting everyone's lives, I decided to make a webpage that can display Covid-19 statistics per state so everyone would be able to view the status of their state in regards to Covid. Simply select the State Statistics button and select your state using your two letter State Code. In addition I provided a button that will display links to Youtube Videos on Covid-19 safety and how you can do your part and stay safe and healthy.
